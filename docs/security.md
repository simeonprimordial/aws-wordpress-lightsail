# Security Review

## Security Controls Implemented

- Linux-based server
- AWS-managed WordPress blueprint
- Static IP assignment
- Updated WordPress installation
- Removal of default content
- Snapshot recovery strategy

---

## Security Best Practices

- Use strong administrator credentials.
- Keep WordPress updated.
- Remove unused themes and plugins.
- Limit administrative access.
- Enable HTTPS after configuring a domain.
- Review snapshots regularly.

---

## Future Security Enhancements

- Configure Let's Encrypt SSL/TLS.
- Register a custom domain.
- Enable AWS CloudFront.
- Protect the application with AWS WAF.
- Implement monitoring and alerting.
